# Blinkit Clone

A full-stack Blinkit Clone built using HTML, CSS, JavaScript, Node.js, Express.js and MySQL. This project allows users to browse grocery products, detect their location, add products to cart and view products dynamically from the database.

## Features

- Dynamic product listing from MySQL database
- Add to Cart functionality
- Product categories (Dairy, Vegetables, Snacks, etc.)
- Search and filter products
- Browser geolocation support
- Responsive UI
- Stock management
- MySQL database integration
- Real-time product fetching using API

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL
- phpMyAdmin
